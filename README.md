### 🚲 Citi Bike Jersey City Data Analysis

A Python-based data analysis project that explores Citi Bike Jersey City trip data. This project demonstrates a complete data analytics workflow, including data collection, cleaning, enrichment, visualization, weather integration, and neighborhood-level geospatial analysis.

## Project Overview

This project includes the following steps:

1. **Download Citi Bike Jersey City trip data**
2. **Clean and preprocess the trip data**
3. **Collect and integrate historical weather data**
4. **Enrich the trip data** with weather and additional features
5. **Create data visualizations** for exploratory analysis
6. **Perform neighborhood-level geospatial analysis**
